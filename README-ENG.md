### **[📂 Web App 2 – Company Document Portal](https://github.com/DarthRevan7/Company-Document-Portal)**
A distributed microservices platform for intelligent document management, built on a **Zero-Trust** security model (Keycloak) and an **Event-Driven** architecture (Apache Kafka).
* **Tech Stack:** Kotlin, Spring Boot, PostgreSQL + **pgvector**, Apache Kafka, MinIO, OpenTelemetry, Langtrace.
* **Core Pillars:** 
  * **AI Integration & RAG:** Automated document metadata enrichment via LLM (Gemini 2.5 Flash) and a "Chat with your Data" module featuring permission-aware, contextual retrieval.
  * **Transactional Outbox Pattern:** Guarantees data consistency between the relational database and the Kafka broker using Debezium.
  * **Full-Stack Observability:** Centralized monitoring of system metrics, distributed logs, and AI pipelines via Prometheus, Loki, Grafana, and Langtrace.
  * **Robust Testing Suite:** Resilient testing strategy (Idempotency, M2M Security, Kafka Consumer) fully implemented using JUnit 5 and **Testcontainers** (Docker).

### **[📝 Web App 1 – "Last Ride" (SPA Web Game)](https://github.com/DarthRevan7/WA1-Ultima-Corsa)**
A real-time full-stack web application centered around a staged subway route planner web game, driven by a solid client-server architecture and dynamic state management.
* **Tech Stack:** React, Vite, Node.js, Express, SQLite, Passport.js, Bootstrap.
* **Core Pillars & Architecture:**
  * **Client-Side State Machine:** Structured game flow management via a React finite state machine that coordinates Setup, Planning, and asynchronous Execution (Stage Runner) phases in real time.
  * **Real-Time Algorithms & Validation:** Integration of routing logic over transit graphs; the server securely validates the integrity of client-submitted paths, evaluates scores, and processes randomized in-game events impacting the game state.
  * **Asynchronous UX & Timers:** Advanced usage of React Hooks to manage a 90-second countdown (with automatic path submission upon expiration) and sequential, asynchronous rendering of trip stages.
  * **Session Management:** Secure user profile authentication and persistent global leaderboards powered by Passport.js with session storage on a relational SQLite database.
